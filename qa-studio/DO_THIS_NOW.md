# ✅ DO THIS NOW - 3 Steps

## Step 1: Open Correct Folder
1. Close VS Code
2. Open VS Code
3. File → Open Folder
4. Select the **`qa-studio`** folder (NOT the parent)
5. Click "Select Folder"

## Step 2: Initialize & Commit
1. Press `Ctrl+Shift+G`
2. Click "Initialize Repository"
3. Click `+` to stage all
4. Type: `Initial commit: QA Studio`
5. Click ✓

## Step 3: Push
1. Click `...` menu
2. Remote → Add Remote
3. URL: `https://github.com/Khushal05220/QA-Studio-Kiro.git`
4. Name: `origin`
5. Click `...` → Push → Force Push

## Done! ✅

Visit: https://github.com/Khushal05220/QA-Studio-Kiro
