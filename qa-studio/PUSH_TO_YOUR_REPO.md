## Option 2: Using Terminal

Open terminal in VS Code (`Ctrl+``) and run:

```bash
# Navigate to project
cd qa-studio

# Initialize git (if not done)
git init

# Add all files

# Commit

# Add your remote
git remote add origin https://github.com/Khushal05220/QA-Studio-Kiro.git

# Push to GitHub

git push -u origin main
```
